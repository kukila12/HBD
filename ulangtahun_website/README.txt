# Website Ulang Tahun (Siap Pakai)

File:
- index.html
- assets/foto-placeholder.jpg

Cara pakai:
1) Buka `index.html` di browser (PC/HP).
2) Ganti foto: letakkan fotomu di folder `assets/` lalu ubah src pada tag <img> di index.html menjadi file fotomu.
3) Ganti lagu: di bagian <audio id="bgMusic" src="...">, ubah URL ke file MP3 kamu sendiri (bisa online atau file lokal).
4) Upload ke hosting mana pun (GitHub Pages, Netlify, Vercel, dll) agar bisa dibagikan.

Catatan:
- Autoplay musik dibatasi di beberapa browser; tekan tombol "Putar Musik" untuk mulai.
- Desain responsive & efek confetti sudah aktif.
